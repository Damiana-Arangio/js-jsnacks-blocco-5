<h1 align="center">Snack Array Methods</h1>

Esercizi di JavaScript dedicati all’utilizzo dei principali **Array Methods**, realizzati per consolidare le basi della manipolazione dei dati in JavaScript.

---

## Obiettivo

- Applicare i principali metodi degli array di JavaScript  
- Manipolare e trasformare array di dati  
- Lavorare con array di oggetti  
- Migliorare la capacità di scegliere il metodo più adatto in base al problema  

Gli esercizi sono pensati per essere eseguiti in console e hanno lo scopo di rafforzare la comprensione dei meccanismi fondamentali del linguaggio.

---

## Tecnologie utilizzate

- HTML5  
- JavaScript
